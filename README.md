# Student Management System (Flask)

A simple **Student Management System** built with **Python + Flask**.
It supports CRUD operations, search, analytics (age + degree stats), a degree bar-chart, and CSV export.

## Features
- Add / View / Edit / Delete students (CRUD)
- Search by **ID / Name / Degree**
- Prevent duplicate Student IDs
- Statistics:
  - Total students
  - Average, Min, Max age
  - Students by degree
- Degree distribution **bar chart**
- Export all students to **CSV**

## Tech Stack
- Python
- Flask
- JSON file storage (`students.json`)
